Введение в микроразметку Schema.org

Даже хорошая семантическая верстка сайта не передаст поисковикам смысловое значение контента страницы. Я расскажу о том, как помочь роботам лучше понимать ваш сайт с помощью форматов семантической разметки данных

Что такое Schema.org?

Schema.org — это стандарт семантической разметки данных в сети, созданный поисковыми системами Google, Bing, Yahoo! и Яндекс.

Семантика

            <div class="article">
              
                <div class="article__title">...</div>
                <div class="article__subtitle">...</div>
              
              <div class="article__description">...</div>
              <div class="article__body">...</div>
              <div class="article__meta">...</div>
            </div>

Семантика

            <main class="article">
              <header class="article__header">
                <h1  class="article__title">...</h1>
                <p   class="article__subtitle">...</p>
              </header>
              <div class="article__description">...</div>
              <div class="article__body">...</div>
              <div class="article__meta">...</div>
            </main>

Семантика

            <main class itemscope itemtype="https://schema.org/Article">
              <header class="article__header">
                <h1 class itemprop="headline">...</h1>
                <p class itemprop="alternativeHeadline">...</p>
              </header>
              <div class itemprop="description">...</div>
              <div class itemprop="articleBody">...</div>
              <div class="article__meta">...</div>
            </main>

Семантика

            <main class="article"
                  itemscope
                  itemtype="https://schema.org/Article">
              ...
            </main>
